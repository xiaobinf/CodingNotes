这里存在重复子问题
可能可以动态规划
```go
func dailyTemperatures(T []int) []int {
    n := len(T)
    dp := make([]int, n)
    for i := n - 2; i >= 0; i-- {
        j := i + 1
        for j < n && T[i] >= T[j] {
            if dp[j] > 0 {
                j += dp[j]
            } else {
                j = n
            }
        }
        if j < n {
            dp[i] = j - i
        }
    }
    return dp
}
```
对于每个元素，我们需要找到它之后的第一个比它大的元素，然后计算它们之间的距离。因此，我们可以从后往前遍历数组，维护一个数组 dp，其中 dp[i] 表示从第 i 天开始需要等待的天数。对于每个元素，我们可以比较它和它后面的元素，如果后面的元素比它大，则 dp[i] = 1，否则我们需要找到后面第一个比它大的元素，然后计算它们之间的距离。
